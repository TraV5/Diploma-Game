Тема проекта: Создание игры: Поход к “Эльбрусу”.

Описание проекта: Создание игры (2D платформера) на базе Godot. Главным героем является робот, включившийся после неизвестных ему событий. Его задачей является выяснение того, что происходит сейчас и кем он был до этих событий.

Использование технологий: • Godot • С# • GitHub

Возможности предоставляющая игра:

Изучение простого по структуре мира (прямолинейного или коридорного)
Решение не сложных головоломок (По возможности математических “Без сложных формул и т.д”)
Чтение небольших диалогов
Выбор сложности (Под большим вопросом, так как представления по разработке нету “Не влияет на головоломки”)
Поиск секретных комнат или пасхалок (Не влияет на продвижение по игре)
Будет реализовано в будущих версиях
